The definite ToDo list app



Hello world! This is an app built using Angular.js.


You can use it to track the tasks you have for the week and set an alarm by double tapping or clicking on each task (it will display a green left border)



Features

It is divided into various components. The most important ones are the Add Task Component, which you can toggle on and off by the Add button from the header; and the Task view component, which shows all the tasks available


There is also routing for this about page, and the use of Input and Output used for the smaller components to deliver change of states.


The task information is stored in a json database (you can find it in the root folder as db.json), so you can refresh the page and the data will remain.



Summary

Overall, it is a very simple app built with the aim of learning the complexities of Angular, fighting off its steep learning curve and building a very simple yet beautiful and useful application to make myself proud.

I hope you enjoy it! thanks for using it
